# Figma Dev Mode 26_4 - Direcciones

Replica estatica en HTML y CSS de la pantalla de inicio de sesion de Jala University.

## Link publicado

https://santiagocard123.github.io/figma-direcciones-replica-26-4/

## Archivos

- `index.html`: estructura HTML.
- `styles.css`: estilos extraidos/aproximados desde la referencia visual.
- `assets/night-bokeh-background.png`: fondo bitmap generado para recrear el asset visual.
- `assets/jala-logo.svg`: aproximacion vectorial del logotipo.
- `evidence/figma-reference-thumbnail.webp`: referencia publica disponible desde Figma.
- `evidence/replica-fullpage.png`: screenshot local del resultado.
- `evidence/published-fullpage.png`: screenshot del sitio publicado.
- `evidence/figma-dev-attempt.png`: evidencia del intento de abrir Dev Mode desde automatizacion.

## Nota de acceso

El enlace publico de Figma permite ver el archivo, pero la cuenta conectada al plugin no tiene acceso completo a Dev Mode (`can_access_full_dev_mode: false`) y el intento automatizado de abrir `mode=dev` devolvio `403 CloudFront`. Por eso las medidas se replicaron desde la referencia visual publica y no desde capas completas exportadas por Dev Mode.
